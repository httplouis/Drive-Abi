Return Instructions

Easy and hassle-free vehicle return process

How to Return Your Vehicle

Follow these simple steps for a smooth return process

{% for step in "1,2,3,4,5,6" %}
{{ step }}
{% if step == "1" %}

Return On Time

Return the vehicle at or before your scheduled return time. Late returns will incur additional charges at an hourly rate.

Note: If you need to extend your rental, contact us at least 24 hours before the scheduled return time.

{% elif step == "2" %}

Refuel the Vehicle

Return the vehicle with a full tank of fuel. We provided it to you with a full tank, and we expect the same on return.

Correct:

Fill up at a nearby gas station before returning

Incorrect:

Returning with less than a full tank (₱100/liter refueling fee applies)

{% elif step == "3" %}

Clean the Interior

Remove all personal belongings and trash. The vehicle should be returned in the same clean condition as when you received it.

  • Remove all trash and personal items
  • Vacuum or shake out floor mats if excessively dirty
  • Wipe down any spills or stains
  • Check under seats and in compartments for belongings

Cleaning Fee: Excessively dirty vehicles will incur a ₱1,500 cleaning fee.

{% elif step == "4" %}

Inspect for Damage

Check the vehicle for any new damage that occurred during your rental period. Report any issues immediately.

Accidents

Report all accidents immediately, no matter how minor

Scratches/Dents

Document any new scratches or dents with photos

Mechanical

Report any mechanical issues or warning lights

{% elif step == "5" %}

Return Location

Return the vehicle to the agreed-upon location. We offer multiple return options:

Office Return

Return to any of our office locations during business hours

FREE

Pickup Service

We'll pick up the vehicle from your location

₱500

Airport Return

Drop off at designated airport locations

₱800
{% elif step == "6" %}

Final Inspection

Our staff will conduct a final inspection of the vehicle. This typically takes 10-15 minutes.

What We Check:

Fuel Level

Mileage

Condition

Damages

All Clear? You'll receive a confirmation email and your security deposit will be released within 3-5 business days.

{% endif %}
{% endfor %}

Important Notes

Late Return Fees

  • 1-2 hours late: ₱500 fee
  • 2-4 hours late: ₱1,000 fee
  • 4+ hours late: Full day rental charge

Damage Charges

  • Minor scratches: ₱2,000 - ₱5,000
  • Dents/Panels: ₱5,000 - ₱20,000
  • Major damage: Actual repair cost

Lost Items

  • Lost key: ₱5,000 replacement
  • Lost documents: ₱1,000 processing
  • Lost accessories: Replacement cost

Perfect Return Bonus

Return the vehicle in perfect condition and earn loyalty points towards your next rental!
